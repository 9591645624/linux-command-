# linux-command-
Excuted linux Command list of the using most imporatant 
cd - chnage file directory 
ls - list file directory 
mkdir - create new empty directory 
pwd - print working directory 
rm -remove file and directory 
rmdir -remove emty file directory 
mv - move and rename file directory 
cp -copy file and directory 
touch - create empty file update the timestamp
cat - concatenate file line by line 
less - view file page by page 
grep - search for specific pattern file and directory 
find - search for file and directory 
df  - display disk space usage 
du - display disk usage and file & directory 
free -  show memory and swap usage 
top - display running process and system resource 
ps - show current process
kill- display terminate process 
chown - change file permission 
chmod - change file owner and group
